Architecture

Caddy (reverse proxy, port 80/443)
  /chat, /voice        -> Channel adapters (optional public)
  /admin               -> Admin App (LAN only)
  /opencode            -> OpenCode UI (LAN only)
  /openmemory          -> Open Memory UI (LAN only)
  /message, /health    -> Gateway
  /channel/*           -> Gateway (defense in depth)

Core: Gateway -> OpenCode -> Open Memory
Admin: Admin App -> Controller (container lifecycle)
Storage: PostgreSQL, Qdrant, Shared FS

Container Management (via API)

POST /admin/containers/list     — list running containers
POST /admin/containers/up       — start a service
POST /admin/containers/down     — stop a service
POST /admin/containers/restart  — restart a service

Extension Lifecycle (via API/CLI)

POST /admin/extensions/request  — queue a plugin for install
GET  /admin/extensions/list     — review queue
POST /admin/extensions/apply    — apply (requires step-up)
POST /admin/extensions/disable  — disable (requires step-up)
