# @openpalm/channel-api

OpenAI and Anthropic compatible API facade channel for OpenPalm. Routes standard LLM API calls to the OpenPalm guardian.

## Endpoints

| Method | Path | Description |
|---|---|---|
| `POST` | `/v1/chat/completions` | OpenAI chat completions |
| `POST` | `/v1/completions` | OpenAI legacy completions |
| `POST` | `/v1/messages` | Anthropic messages |
| `GET` | `/v1/models` | List available models |
| `GET` | `/health` | Health check |

## Usage

Point any OpenAI- or Anthropic-compatible client at `http://localhost:8182`.

## Environment variables

| Variable | Purpose |
|---|---|
| `PORT` | HTTP listen port (default: `8182` in registry overlay) |
| `GUARDIAN_URL` | Guardian forwarding target |
| `CHANNEL_API_SECRET` | HMAC secret (auto-generated by admin) |
| `OPENAI_COMPAT_API_KEY` | Optional Bearer / x-api-key auth for incoming requests |

## Registry component

`registry/components/api/compose.yml` -- install via admin API:

```bash
curl -X POST http://localhost:8100/api/registry/api/install \
  -H "x-admin-token: $ADMIN_TOKEN"
```

Streaming is not supported (returns `400`).
