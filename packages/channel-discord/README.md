# @openpalm/channel-discord

Discord webhook adapter channel for OpenPalm. Receives incoming webhook POST requests and forwards them as signed channel messages to the guardian.

## Endpoint

| Method | Path | Description |
|---|---|---|
| `POST` | `/discord/webhook` | Receive Discord webhook payload |
| `GET` | `/health` | Health check |

## Payload format

```json
{
  "userId": "123456789",
  "text": "hello",
  "channelId": "...",
  "guildId": "...",
  "username": "..."
}
```

`userId` is prefixed with `discord:` before forwarding to the guardian.

## Environment variables

| Variable | Purpose |
|---|---|
| `PORT` | HTTP listen port (default: `8184` in registry overlay) |
| `GUARDIAN_URL` | Guardian forwarding target |
| `CHANNEL_DISCORD_SECRET` | HMAC secret (auto-generated by admin) |

## Registry overlay

`registry/channels/discord.yml` — install via admin API:

```bash
curl -X POST http://localhost:8100/admin/channels/install \
  -H "x-admin-token: $ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"channel": "discord"}'
```
